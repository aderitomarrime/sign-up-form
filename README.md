## Overview

This project consists of a sign-up form that collects user information such as first name, last name, email, phone number, and password with confirmation.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/sign-up-form/)


## Features

- Minimal interface  
- Collects user information through form inputs
- Simple form validation


## Technologies Used

- HTML5
- CSS3


## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/sign-up-form.git
```

2. Navigate to the project 
```sh
cd sign-up-form
```
3. Open index.html in your browser


## How To Use

Once the website is open, fill in the inputs and then click the "Create Account" button.
